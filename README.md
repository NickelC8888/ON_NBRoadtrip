# Road Trip Planner

Curated 3–7 day round trips from Toronto for families with kids and a dog.

## Features
- **5 curated routes** — Tobermory, Muskoka, Prince Edward County, Ottawa/Gatineau, and a 7-day Eastern Canada Grand Circuit
- **Interactive Leaflet maps** — route polyline with clickable stop markers
- **Dog-friendly POI & trails** — accurate dog policy flags on every attraction and trail
- **Restaurants** — best places to eat with dog-friendly patio notes
- **Pet-friendly lodging** — vetted accommodations for families with dogs
- **Samoyed heat-safety guidance** — breed-specific notes on all trails
- **Packing list** — 75-item interactive checklist across 6 categories
- **Google Maps link** — open the full route in Google Maps (7-day trip)
- **Print trip plan** — clean print view of the selected trip

## Tech Stack
- React 18 + Vite
- Tailwind CSS v3
- react-leaflet v4 + Leaflet (OpenStreetMap, no API key required)
- Radix UI Tabs
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```
